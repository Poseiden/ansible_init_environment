## 环境初始化脚本

#### Ansible 环境初始化
`ansible-playbook -i your_host_path java8.yml`

#### Java 环境初始化
`ansible-playbook -i your_host_path java8.yml`

#### Jenkins 环境初始化
`ansible-playbook -i your_host_path jenkins.yml`

#### Nodejs 初始化
`ansible-playbook -i your_host_path geerlingguy.nodejs.yml `

#### Nginx 初始化
`ansible-playbook -i your_host_path nginx.yml `
